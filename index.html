这是什么？一个免费的在线注意力测试（attention span test）：60 秒、3 个像素小游戏， 基于 Posner & Petersen 的三大注意力网络与 ANT 范式，测量你的警觉、定向与执行控制， 输出六种注意力类型之一（猎豹 / 鹰眼 / 堡垒 / 指挥官 / 均衡 / 漫游者）与个性化建议。 无需注册登录，所有计算在你的浏览器本地完成。

AttentionOS 同时是一个开源的注意力观测层：macOS 桌宠会根据你的真实专注数据改变行为， MCP 接口让 AI Agent 在打断你之前先查看你的专注状态。
